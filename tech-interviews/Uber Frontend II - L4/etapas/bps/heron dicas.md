## 7 Critérios de avaliação objetivos da BPS

1. Coleta de Requisitos
2. Design da Solução
3. Análise de Complexidade
4. Implementação
5. Qualidade de Código
6. Casos de Teste
7. Debugging
## Passo a passo heron

1. Levantamento de Requisitos, perguntas clarificadoras
2. Assinatura do método
3. Test cases incluindo edge cases
4. Discutir abordagens possíveis incluindo detalhes de implementação e análise de complexidade
5. Escrever o passo a passo da sua implementação
6. Fazer a implementação rodando parcialmente o código até completar a função inteira (lembrando de seguir boas práticas, clean code e DRY)
7. Rodar
8. Debuggar, se necessário, de forma estruturada
