# Contexto Uber

Esta pasta agrupa **termos, ferramentas e práticas** que aparecem no processo seletivo da Uber (e em comunicações típicas de engenharia), para consulta rápida sem depender só dos e-mails.

## Tópicos documentados

| Pasta / arquivo | O quê |
|-----------------|--------|
| [`hackerrank/`](hackerrank/) | Uso do **HackerRank** nas entrevistas (IDE compartilhada, whiteboard, fluxo com Zoom). |
| [`uber-stack/`](uber-stack/) | Stack **frontend web** documentada publicamente + link do [meetup no YouTube](https://www.youtube.com/watch?v=EGHHaOgUzPU) e limites da análise sem transcrição. |

Novos temas podem vir como subpastas (`context/<tema>/README.md`), por exemplo: BPS, Talent Coordinator, iCIMS, Candidate Prep Pack, etc.
